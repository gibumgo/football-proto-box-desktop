# Football Proto Box Desktop

축구 데이터를 크롤링하고 분석하는 툴을 만들던 중, 이를 데스크탑 앱으로 좀 더 편리하게 사용해보고 싶다는 생각에서 시작된 프로젝트입니다.

거창한 플랫폼을 목표로 하기보다는 파편화된 축구 관련 기획들을 데스크탑 환경에서 효율적으로 관리하고 시각화하려는 개인적인 니즈를 담고 있습니다.

## 주요 동기
- **크롤러의 확장**: 단순 데이터 수집에서 벗어나 수집된 데이터를 직접 제어하고 시각화할 수 있는 인터페이스의 필요성
- **데스크탑 경험**: 웹 브라우저보다 더 로컬 환경에 밀착된 빠른 접근성과 데이터 관리
