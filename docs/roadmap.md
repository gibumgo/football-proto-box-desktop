# Proto Box 개발 로드맵

## 📋 프로젝트 개요
축구 프로토 베팅 데이터를 분석하고 가설 기반 승률 시뮬레이션을 제공하는 전문가용 데이터 뷰어 & 분석 도구

---

## 🎯 현재 구현 상태

### ✅ 완료된 항목

| 카테고리 | 항목 | 상태 | 설명 |
|---------|------|------|------|
| **디자인 시스템** | VS Code Dark+ 테마 | ✅ | 어두운 배경, 네온 액센트 색상 |
| **디자인 시스템** | 디자인 토큰 | ✅ | `theme.ts`, `layout.ts`, `tokens.ts` |
| **레이아웃** | AppLayout | ✅ | Sidebar + Topbar + Content 구조 |
| **레이아웃** | Sidebar | ✅ | 내비게이션 메뉴 (Dashboard, Analysis, Tools, Settings) |
| **레이아웃** | Topbar | ✅ | 통계 바, Signal/Density 토글 |
| **페이지** | Dashboard | ✅ | 전체 요약, 통계 카드, 차트 |
| **페이지** | Matches (기본) | ✅ | 경기 목록 테이블, 결과 네온 효과 |
| **도메인 모델** | DDD 리팩토링 | ✅ | Match, MatchInfo, MatchOdds, MatchResult |
| **비즈니스 로직** | SignalLogic | ✅ | 정배 붕괴/역배 적중 판단 |
