# 최근 기능 구현 목록 (2025-12-18 기준)

본 문서는 최근 6시간(2025-12-17 오후 ~ 2025-12-18 오전) 동안 구현된 주요 기능 및 변경 사항을 정리한 문서입니다.

## 1. 팀 프로필 페이지 (Team Profile Page)

특정 팀의 상세 정보를 확인하고 최근 전적 및 배당 흐름을 분석할 수 있는 **팀 프로필 페이지**가 구현되었습니다.

### 주요 기능
- **전적 요약 (Summary Stats)**
  - 최근 경기 결과에 따른 승/무/패 스트릭(Streak) 표시
  - 전체 승/무/패 횟수 집계
  - 경기당 평균 득점/실점
  - 2.5 오버(Over 2.5) 비율 및 클린조(Clean Sheet) 비율 표시

- **경기 기록 리스트 (Match History List)**
  - 날짜, 리그, 홈/원정 구분, 경기 결과(승/무/패), 스코어 표시
  - 배당 정보(승/무/패 배당) 표시 및 결과 적중 여부 하이라이트
  - 페이지네이션 (Pagination) 지원

## 2. 레이아웃 및 필터 개편 (Layout & Filter Refactoring)

사용자 경험(UX) 개선을 위해 필터 영역의 레이아웃을 재설계하고 컴포넌트 구조를 개선했습니다.

### 변경 사항
- **필터 사이드바 구조 개선**
  - 기존의 가로/세로 혼합 배치에서 **정보 위계가 명확한 수직 스택(Vertical Stack)** 구조로 변경
  - `TeamMatchFilters` 컴포넌트 재설계:
    - **경기 종류(Match Type)** 섹션과 **배당 선택(Odds Selection)** 섹션을 명확히 분리
    - 각 섹션 간 구분선(Divider) 추가 및 간격(Gap) 조정 (24px)
  
- **UI/UX 개선**
  - **배당 버튼 스타일링**: 배당 버튼이 좁은 영역에서도 자연스럽게 배치되도록 Grid 레이아웃 적용 (텍스트 줄바꿈 방지)
  - **섹션 헤더**: 각 필터 그룹에 명확한 소제목(Typography SM, Bold) 적용하여 가독성 강화
  - **불필요한 중첩 스타일 제거**: 사이드바와 내부 필터 컨테이너 간의 중복된 패딩 및 배경색 제거로 깔끔한 디자인 구현

## 3. 기술적 변경 사항 (Technical Changes)

- **컴포넌트 분리**: `TeamMatchFilters`, `MatchTypeFilter`, `OddsFilter` 등으로 역할을 분리하여 유지보수성 향상
- **도메인 테마 활용**: `COLORS`, `TYPOGRAPHY` 상수 활용을 통한 일관된 디자인 시스템 적용
