# 🎨 Flashscore UI Redesign Proposals

현재의 "국가/리그 선택" UI가 좁고 답답하다는 피드백을 반영하여, 3가지의 새로운 레이아웃 방향성을 제안합니다.

목표: **Premium, Dynamic, Intuitive**

---

## 옵션 1: Mac Finder 스타일 (Modern Split View)
> **"빠르고 익숙한 계층 탐색"**

기존의 좌우 분할 방식을 유지하되, 시각적 완성도와 사용성을 대폭 개선한 형태입니다. MacOS의 Finder나 Notion의 컬럼 뷰와 유사합니다.

### 특징
- **높이 확장**: 고정된 300px 대신 화면 높이에 맞춰 유동적으로 확장 (`flex-grow`).
- **Visual Feedback**:
    - 선택된 항목에 **Glassmorphism** 효과 (투명한 배경 + 블러).
    - 국가명 옆에 **국기(Flag Emoji)** 표시로 시각적 인지 속도 향상.
    - 리그명 옆에 **리그 아이콘** (또는 플레이스홀더) 배치.
- **Sticky Search**: 각 컬럼 상단에 고정된 검색바 배치.
- **Animation**: 선택 시 오른쪽 컬럼이 부드럽게 슬라이드되어 나타나는 효과.

### 장점
- 많은 데이터(국가/리그)를 빠르게 훑어보기에 가장 효율적.
- 마우스 이동 거리가 짧음.

## 옵션 2: 넷플릭스 스타일 (Visual Card Grid)
> **"탐색하는 즐거움과 시각적 몰입감"**

텍스트 리스트 대신 **카드(Card)** 형태의 그리드 레이아웃을 사용합니다.

### 특징
- **Step-by-Step**:
    - **1단계 (국가)**: 화면 가득 국기/국가명이 적힌 카드가 그리드로 펼쳐짐.
    - **2단계 (리그)**: 국가 클릭 시, 해당 국가의 리그들이 카드(또는 타일) 형태로 전환됨.
- **Hero Header**: 상단에 현재 선택된 경로(Breadcrumb)를 크게 표시 (예: 🇬🇧 England > 🏆 Premier League).
- **Hover Effects**: 카드 호버 시 줌인(Scale Up) 및 글로우 효과.

### 장점
- **심미성 최상**. 가장 "프리미엄" 해 보임.
- 터치 인터페이스나 넓은 모니터에서 시원시원한 느낌.

### 단점
- 스크롤이 길어질 수 있음 (검색 기능이 필수).
- "빠른 변경"을 위해서는 뒤로가기 버튼을 눌러야 함.

## 옵션 3: 커맨드 센터 스타일 (Global Search / Command Palette)
> **"키보드 중심의 파워 유저 경험"**

복잡한 클릭 없이 단순하고 강력한 **하나의 검색창**에 집중합니다. (Raycast, VS Code 스타일)

### 특징
- **통합 검색창**: "eng prem..." 이라고 치면 알아서 **[국가: England] + [리그: Premier League]** 조합을 찾아줌.
- **자동 완성**: 입력에 따라 추천 국가/리그가 드롭다운으로 펼쳐짐.
- **미니멀리즘**: 평소에는 검색창 하나만 있다가, 선택 완료 시 결과 카드만 예쁘게 표시.

### 장점
- **가장 빠름**. 목록을 뒤질 필요가 없음.
- 화면 공간을 가장 적게 차지함.

### 단점
- "어떤 나라가 있는지" 둘러보고 싶은 사용자에게는 불친절함 (탐색성 부족).

## 💡 추천: 하이브리드 "Modern Split View" (옵션 1 개선안)

데스크탑 앱의 생산성과 탐색의 용이성을 고려했을 때, **옵션 1(Finder 스타일)**을 베이스로 하되, 옵션 2의 **시각적 요소(Flag, Icons)**를 결합하는 것을 추천합니다.

### 제안 디자인 디테일
1.  **컨테이너**: 전체 높이 사용, 반투명 유리 재질 배경.
2.  **리스트 아이템**:
    - `Py-2 Px-4` (넉넉한 패딩).
    - `Flex row`로 아이콘과 텍스트 정렬.
    - **Active State**: 네온 그린/사이버 펑크풍의 테두리 및 배경 하이라이트.
3.  **오른쪽 패널 (리그)**:
    - 국가 선택 전에는 "Select a Country" 라는 멋진 일러스트/아이콘 표시.
    - 리그 선택 시 리그 로고(없으면 기본 트로피)와 함께 강조 표시.
